# Micro Machines Components – Website

Official website for Micro Machines Components, Chennai.

## Pages
- `index.html` — Homepage
- `contact.html` — Contact Operations
- `quote.html` — Request a Technical Quote
- `thank-you.html` — Form submission confirmation

## Setup
1. Create a free account at formspree.io
2. Create a new form and copy your endpoint ID
3. In `contact.html` and `quote.html`, replace `YOUR_FORM_ID` with your actual Formspree ID

## Deployment
Hosted on GitHub Pages, domain managed via Squarespace DNS.
